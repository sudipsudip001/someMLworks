To run the Agent:

1. uv sync <br>
2. activate your virtual environment: <br>
   `source .venv/bin/activate`
3. goto https://aistudio.google.com/api-keys and (create a project) and (API key) <br>
   duplicate .env.example and name it .env: <br>
   replace `your_google_api_key_here` with your API key <br>
4. run the program using 'python main.py' <br>
